Visualizations
There are several interactable element present on the visualization page, which are:
  • The header with the submit, herein you can
    • Submit whoms data to use
    • Select which list to use
      • Completed: Completed Entries
      • On-Hold: Entries, which momentarily have been suspended
      • Dropped: Entries, which have been suspended. Most likely for an indefinite period of time
      • PTW:Entries which are Planned to Watch
  • The BarChart
    • The bars of the BarChart, which influence which genre is represented in the HeatMap
    • The options besides the BarChart
  • The HeatMap
    • The Slider, which influences the 10 year range of the HeatMap
    • The squares within the Heatmap, which influence the agenda beside the HeatMap
  • The LineChart
    • The lines of the LineChart, which influence which genre is represented in the HeatMap
    • The Checkboxes beside the LineChart
    • The CheckAll/unCheckAll buttons
